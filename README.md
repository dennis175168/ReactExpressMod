## React Express Mod 

## Prerequisites

- [Node JS](https://nodejs.org/en/download/)

## To try this sample

1) Clone the repository

    ```bash
    git clone 
    ```


1) In a terminal, navigate to this package

1) Install modules

    ```bash
    npm install --save
    ```
1) navigate to this package and start demo 

    ```bash
    npm start 
    ```
